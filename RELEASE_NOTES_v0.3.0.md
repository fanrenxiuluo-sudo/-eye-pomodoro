## 🎉 EyeTimer v0.3.0 — 护眼全面升级

### 🧠 权威护眼休息指导
基于美国眼科学会 (AAO) 20-20-20 法则、眼科协会建议，休息时显示专业护眼提示：
- **短休息**：8 条快速眼部放松动作（眨眼练习、远近对焦、掌心热敷等）
- **长休息**：8 条完整护眼操 + 身体活动（眼保健操、肩颈放松、环境检查等）
- 每条提示附带科学来源和建议耗时

### ⏰ 醒目提醒系统
专注时间到后，提醒不再被忽略：
- 全屏毛玻璃模态弹窗 + 渐变色 + 脉冲动画
- 任务栏持续闪烁直到用户点击
- 提示音增强：音量提升、旋律更复杂、重复 5 次

### 🎨 UI 全面焕新
